# ✨ Fitur

| # | Fitur | Description |
|---|-------|-------------|
| 1 | 🖥️ **VPS Overview** | Semua server dengan status online/offline |
| 2 | 💻 **SSH Terminal** | Terminal interaktif (xterm.js) via WebSocket |
| 3 | 🐳 **Container Management** | List, start, stop, restart container Docker |
| 4 | 📋 **Container Logs & Stats** | Log dan resource usage per container |
| 5 | 📊 **System Logs** | Lihat syslog, auth, kernel, docker logs |
| 6 | ⚡ **Quick Commands** | One-off commands dengan preset buttons |
| 7 | 📈 **Resource Monitoring** | CPU, RAM, Disk, Load Average visual bar |
| 8 | 👥 **User Management** | Create, edit, delete users |
| 9 | 🔐 **RBAC** | Admin, Operator, Viewer dengan akses per-VPS |
