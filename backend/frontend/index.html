shield

Admin

Full system access. Can manage VPS, users, containers, view audit logs, and use terminals. Can invite new members.
build

Operator

Can view VPS, use SSH terminals, run commands, and manage containers. Cannot add/edit/delete VPS or manage users.
visibility

Viewer

Read-only access. Can view VPS instances, system info, resources, and container status. No terminal or command execution.
