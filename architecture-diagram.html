VPS Dashboard Architecture

Multi-Server Management Platform — Docker Compose Deployment

Frontend Layer

  • • Single Page App (Vanilla JS)
  • • xterm.js for SSH terminal
  • • Dark theme responsive UI
  • • WebSocket for real-time I/O
  • • No build step — served as static files

Backend Layer

  • • FastAPI async web framework
  • • Paramiko SSH connection pooling
  • • WebSocket terminal via PTY channels
  • • Cookie-based session authentication
  • • REST API for CRUD + monitoring

Deployment

  • • Single Docker container
  • • Docker Compose for one-command deploy
  • • Persistent volume for VPS configs
  • • SSH keys mounted read-only from host
  • • ~80MB image (Python 3.12 slim)
